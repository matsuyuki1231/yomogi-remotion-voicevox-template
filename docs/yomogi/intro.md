Welcome to **YOMOGI Server**！(以下、「よもぎ鯖」と表記)

よもぎ鯖のついての情報を初心者の方でも分かるようにお届けしていきます。

:::note

- エディション : `統合版`
- 参加費 : `無料`
- ジャンル : `生活＆経済` `ミニゲーム`
- ミニゲームの種類 : `マイクラ人狼` `銃撃戦` `青鬼`

よもぎサーバーはMinecraft統合版で参加できるサーバーです。Java版ではご利用になれませんのでご注意ください。  
各種システムが統合版向けに最適化されているため、クロスプレイ鯖にありがちな「統合版だと操作しにくい...」という問題は起こりません。  
24時間参加できる生活・経済サーバーと、毎週土曜日の21:30から遊べるマイクラ人狼イベントの2種類のサービスを提供しています。  

# 🌃生活・経済サーバー  
24時間あそべる生活・経済サーバーです。近年台頭している国家形式ではなく、共通の生活ワールドに土地を購入して自分だけの家や店を建築することができます。  
採掘や農業、木こり、釣りなどもでき、作業勢にもおすすめです。  
さらに、他のサーバーには見られない独自の機能も盛りだくさん！飽きずにずっと楽しめます。  

 - 会社制度で本格的な会社経営が楽しめる！  
 - 近距離VCで近くの人としゃべろう！  
 - チェストショップで自分が得たアイテムを販売しよう  
 - 釣りで釣れる魚は275種類！バニラにはない魚がたくさん釣れます  
 - ガチャを引いて様々なレアアイテムをゲットしよう！  
 - 島を作って国家鯖のような生活もできます  
 - 車に乗って生活ワールドを駆け回ろう！

生活サーバーについての詳細は、living配下にあります  

# 🐺マイクラ人狼イベント  
毎週土曜日の21:30から開催している、Minecraftの中で人狼ゲームが遊べるイベントです！  
人狼ゲームの中心要素である会議・投票を行いながら、弓や狙撃銃を使ったPvPで相手の陣営の全滅を目指します。  
サーバー主催者や他の参加者によるサポートも充実しており、初めて参加する人でも安心して遊べます！  
遊べる役職は驚異の41種類！カードの人狼ゲームにある基本役職に加えて、このサーバー独自の役職も充実しています。  

マイクラ人狼イベントについての詳細は、wolf配下にあります

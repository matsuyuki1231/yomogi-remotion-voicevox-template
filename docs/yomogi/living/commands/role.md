# 役職制度

よもぎサーバーには、原木の収集や鉱石採掘など、特定の活動を支援する「役職制度」があります。  
全プレイヤーは、あらかじめ用意された役職のうちのどれか一つを選択することができます。  

## 役職の種類

### 鯖民
初期役職です。特に効果はありません。

### 木こり
原木や木材を採掘した際に5YGと、低確率でエンチャント本を得ることができます。  

### 採掘者
石を採掘した際に2YGを、鉱石を採掘した際に3-20YGを得ることができるほか、これらのブロックの採掘時に低確率でエンチャント本を入手できます。  
2026年現在、各鉱石を採掘した際に得られるYGは以下の通りです。

| 鉱石 | YG |
|:---:|:---|
| 石炭鉱石 | 3YG |
| 鉄鉱石 | 5YG |
| 金鉱石 | 5YG |
| ラピスラズリ鉱石 | 7YG |
| レッドストーン鉱石 | 3YG |
| ダイヤモンド鉱石 | 20YG |

得られるYGの量は、採掘したブロックの数でカウントします。  
たとえば、ラピスラズリ鉱石は1つのブロックから複数のラピスラズリがドロップしますが、ドロップするラピスラズリの数にかかわらず、1回の採掘で得られるYGは7YGです。

### 農家
スイカとかぼちゃを採掘した際に3YGを得ることができるほか、小麦、サトウキビ、スイカ、かぼちゃの採掘時に低確率でエンチャント本を入手できます。  

### エンチャンター
エンチャント本を用いたエンチャントができるほか、全固形ブロックの採掘時に低確率でエンチャント本を入手できます。  

## 役職変更
役職は、/roleコマンドまたはよもぎ端末(ymg-device)の「役職変更」で何度でも変更することができます。  
変更料金は無料で、クールタイムもありません。

## 「鯖民」役職の非推奨性

サーバー初参加後は全員の初期役職が「鯖民」となっていますので、サーバー参加後に役職を変更してください。「鯖民」のままですと、いずれのブロックを採掘しても報酬が得られません。

# サーバーアドレスを入力して参加する

サーバーアドレスを入力することによる生活サーバーの参加方法について説明します。

:::tip このページの要約
サーバーアドレスは「ymg24.org」、ポートは「19132」です。
:::

:::info この方法のメリット
 - 生活サーバーに直接アクセスするため、ポータルサーバーが停止していても参加できます
 - 経由するサーバーの数が少ないため、接続が安定しやすいです
 - スマートフォンやWindows PCから参加する際は、この方法が最速です
:::

:::caution この方法のデメリット
- Nintendo Switchで参加する際は、少々特殊な操作が必要になります
- 生活サーバーとマイクラ人狼イベントの両方に参加される場合は、それぞれでサーバーアドレスの入力が必要になります
:::

## Minecraftサーバーに参加する

➀ Minecraftの統合版を起動し、「プレイ」をクリックします  
②「サーバー」をクリックします  
③ 画面左上の「サーバーを追加」をクリックします

:::caution 注意
Nintendo Switchの方は「サーバーを追加」ボタンがありません。  
サーバーの追加方法については https://docs.ymg24.org/docs/living/faq/device をご覧ください。
  :::

![サーバーを追加ボタンの位置を示したスクリーンショット](../img/how_to_join_1.png)

④ サーバー名に好きなサーバー名を、サーバーアドレスに「ymg24.org」、ポートに「19132」と入力してください。  

![サーバー追加欄に入力すべき項目が書かれたスクリーンショット](../img/how_to_join_2.png)

⑤ 「追加してプレイ」をクリックします  
⑥ サーバーに参加できます　　

:::caution 注意
サーバーに参加できない場合は https://docs.ymg24.org/docs/living/faq/joining-faults をご覧ください。
  :::
